# Based Marks Chrome Extension
This is an extension for chrome browser, it exports the browser bookmarks in json file format. importing bookmarks is done by json file format as well.

Since this extension is not deployed in the chrome web store in order to try it please follow these steps:

1. clone this repo.
2. go to your chrome>extensions.
3. enable `developer` mode.
4. in the top left corner click `load unpacked` button.
5. then simply browse to the directory you pulled this repo to and then click ok it will load the extension and you can use it.

## Important Notes:
- This extension is tested on chrome browser on a linux OS.
- This is the base design of an extension. meaning its for practicing how an extension is structured and works behind the scene.
- This repo is read only but it is open source feel free to use my code.
- The firefox extension follows the same routine. but there are only a few little changes are needed for it to work.
- This is only for practice and it might not be perfect.
